# LearnPython — Платформа для изучения Python

![Python Logo](https://www.python.org/static/community_logos/python-logo-generic.svg)

**LearnPython** — это современная геймифицированная образовательная платформа, предназначенная для обучения программированию на языке Python. Проект сочетает в себе теоретические уроки, интерактивные квизы и встроенную среду выполнения кода прямо в браузере.

## 🚀 Основные возможности

- **26 структурированных уроков**: От основ ("Что такое Python?") до продвинутых тем ("Алгоритмы" и "Отладка").
- **Геймификация**: Система уровней и опыта (XP), отслеживание прогресса пользователя.
- **Интерактивный интерпретатор**: Возможность писать и запускать Python-код в браузере благодаря технологии **Pyodide** и редактору **Monaco Editor**.
- **Мультиязычность**: Полная поддержка русского (`ru`), казахского (`kk`) и английского (`en`) языков.
- **Гибкая авторизация**: Регистрация через Email (с подтверждением), а также быстрый вход через Google и GitHub (OAuth).
- **Личный кабинет**: Управление профилем, просмотр достижений и прогресса по урокам.
- **Модуль для учителей**: Специальный интерфейс для управления группами студентов и мониторинга их успехов.
- **Админ-панель**: Управление пользователями и контентом платформы.

## 🛠 Технологический стек

- **Backend**: Python, Flask, Jinja2
- **Database**: SQLite (с оберткой, имитирующей Firebase API)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap
- **Tools**: 
  - [Pyodide](https://pyodide.org/) — Python в WebAssembly.
  - [Monaco Editor](https://microsoft.github.io/monaco-editor/) — мощный редактор кода (как в VS Code).
  - [Authlib](https://docs.authlib.org/) — для OAuth авторизации.
  - [Flask-Mail](https://pythonhosted.org/flask-mail/) — для отправки системных уведомлений.

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <url-вашего-репозитория>
```

### 2. Настройка виртуального окружения
```bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/macOS
# или
.venv\Scripts\activate     # Для Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка окружения
Создайте файл `.env` в корневой директории и укажите необходимые ключи:
```ini
HF_API_TOKEN=API-ключ-получить-на-huggingface.co
```

### 5. Инициализация базы данных
Если файл `app.db` отсутствует, запустите скрипт миграции:
```bash
python migrate_db.py
```

### 6. Запуск приложения
```bash
python app.py
```
Приложение будет доступно по адресу `http://localhost:5000`.

## 📁 Структура проекта

- `app.py` — Основной файл приложения Flask.
- `static/` — Статические файлы (CSS, JS, изображения, Pyodide, Monaco).
- `templates/` — Шаблоны Jinja2 (интерфейс пользователя, админа и учителя).
- `translations.json` — Файл с переводами интерфейса.
- `quizzes.json` — Данные квизов и вопросов.
- `app.db` — База данных SQLite.
- `requirements.txt` — Список зависимостей Python.

## 🌍 Интернационализация

Для редактирования переводов используйте файл `translations.json`. При добавлении новых квизов убедитесь, что они добавлены в `quizzes.json` и обработаны скриптом `build_quizzes_i18n.py` для корректного отображения на разных языках.
