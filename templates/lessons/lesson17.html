Цикл while — самый универсальный цикл в Python. Он выполняет блок кода снова и снова, пока условие остаётся истинным.

⚙️
Синтаксис while
while условие:
    # тело цикла
    # изменяем переменные, влияющие на условие

Условие проверяется перед каждой итерацией. Как только оно становится ложным, цикл завершается.

🔢
Пример: счётчик
count = 1
while count <= 5:
    print(f"Итерация {count}")
    count += 1   # обязательно увеличиваем счётчик

Вывод:

Итерация 1
Итерация 2
Итерация 3
Итерация 4
Итерация 5

Если забыть изменить count, цикл станет бесконечным.

⌨️
Пример: пока пользователь не введёт 'стоп'
ответ = ''
while ответ != 'стоп':
    ответ = input("Скажи что-нибудь (или 'стоп' для выхода): ")
    if ответ != 'стоп':
        print(f"Ты сказал: {ответ}")

Цикл продолжается, пока пользователь не введёт "стоп".

🔀
Вложенные циклы while
i = 1
while i <= 3:
    j = 1
    while j <= 3:
        print(f"{i} * {j} = {i*j}")
        j += 1
    i += 1
    print("---")

Такой код выводит таблицу умножения для чисел 1–3.
