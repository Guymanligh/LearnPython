Цикл for в Python используется для перебора элементов последовательностей (строк, списков, кортежей, словарей) или диапазонов чисел, созданных функцией range().

📋
Синтаксис цикла for
for переменная in последовательность:
    # тело цикла

На каждой итерации переменная принимает следующее значение из последовательности.

🔢
Функция range()

range(stop) — генерирует числа от 0 до stop-1.

for i in range(5):
    print(i)   # 0 1 2 3 4

range(start, stop) — числа от start до stop-1.

for i in range(2, 6):
    print(i)   # 2 3 4 5

range(start, stop, step) — с заданным шагом.

for i in range(1, 10, 2):
    print(i)   # 1 3 5 7 9
📚
Перебор элементов коллекций

Список:

fruits = ["яблоко", "банан", "апельсин"]
for fruit in fruits:
    print(fruit)

Строка:

word = "Python"
for char in word:
    print(char)

Словарь (по ключам):

person = {"name": "Анна", "age": 25}
for key in person:
    print(key, person[key])
🔀
Вложенные циклы for
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} * {j} = {i*j}")
    print("---")

Так можно обрабатывать таблицы, матрицы или любые комбинации значений.
