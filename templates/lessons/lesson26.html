Алгоритм — это последовательность шагов для решения задачи. Python позволяет легко реализовывать различные алгоритмы: от простого поиска до рекурсивных вычислений.

🔍
Линейный поиск

Проходит по всем элементам последовательности, пока не найдёт нужный.

def linear_search(arr, target):
    for i, value in enumerate(arr):
        if value == target:
            return i          # возвращаем индекс
    return -1                 # не найдено

numbers = [3, 7, 2, 9, 5]
print(linear_search(numbers, 9))   # 3

Сложность: O(n). Работает с любыми неотсортированными данными.

⚡
Бинарный поиск (только для отсортированных массивов)

Делит отрезок пополам и сравнивает средний элемент с искомым.

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

sorted_numbers = [1, 3, 5, 7, 9, 11]
print(binary_search(sorted_numbers, 7))   # 3

Сложность: O(log n). Значительно быстрее линейного на больших данных.

🫧
Сортировка пузырьком (Bubble Sort)

Многократно проходит по массиву, меняя соседние элементы, если они стоят в неправильном порядке.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n-1):
        swapped = False
        for j in range(n-1-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:        # оптимизация: массив уже отсортирован
            break
    return arr

unsorted = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(unsorted))   # [11, 12, 22, 25, 34, 64, 90]

Сложность: O(n²) в худшем случае, но проста для понимания.

🔄
Рекурсия: факториал

Функция вызывает саму себя для решения подзадачи.

def factorial(n):
    if n <= 1:          # базовый случай
        return 1
    return n * factorial(n - 1)

print(factorial(5))     # 120

Важно всегда иметь базовый случай, иначе рекурсия станет бесконечной.

🐚
Рекурсия: числа Фибоначчи
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 0, 1, 1, 2, 3, 5, 8, 13...
for i in range(8):
    print(fibonacci(i), end=' ')   # 0 1 1 2 3 5 8 13

Этот рекурсивный способ неэффективен для больших n (экспоненциальная сложность), но нагляден.
