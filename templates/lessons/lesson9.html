Создание переменной

С помощью знака = число или строку можно поместить в переменную - это называется заданием значения переменной.

bones = 3

bones - имя переменной, 3 - значение переменной

Числовое и строковое значение
Числовое значение

Чтобы задать переменной числовое значение, введите имя переменной, знак = и затем число.

age = 25
Строковое значение

Чтобы задать переменной строковое значение, введите имя переменной, знак = и затем строку в кавычках.

dogs_name = 'Bruno'
Печать переменной

Команда print нужна, чтобы печатать текст на экране. С помощью этой команды можно узнать значение переменной.

print(bones)
3

print(dogs_name)
Bruno
Изменение значения переменной

Чтобы изменить значение переменной, просто задай ей новое значение.

gifts = 2
print(gifts)
2

gifts = 3
print(gifts)
3
Использование переменных

С помощью знака = можно присвоить одной переменной значение другой.

  1. Задаем значения

    Эти команды дают переменной rabbits значение 5, затем дают это же значение переменной hats.

    rabbits = 5
    hats = rabbits
  2. Печатаем значения

    Чтобы вывести на экран значение двух переменных, укажи их в скобках после команды print, поставив между ними запятую.

    print(rabbits, hats)
    5 5
  3. Изменяем значение rabbits

    Если изменить значение rabbits, значение hats останется прежним.

    rabbits = 10
    print(rabbits, hats)
    10 5
Ключевые концепции
  • Переменная - это контейнер для хранения данных
  • Имена переменных должны начинаться с буквы
  • Значения могут быть числами или строками
  • Одна переменная может использоваться в другой
  • Значение переменной можно изменить в любой момент
Попробуй в интерпретаторе

Открой онлайн-интерпретатор и попробуй эти примеры:

  • name = "Алиса"
  • age = 12
  • print(name, "сейчас", age, "лет")
  • age = age + 1
  • print("Через год", name, "будет", age, "лет")
