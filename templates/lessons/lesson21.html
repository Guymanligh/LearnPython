Функции — это многократно используемые блоки кода, которые выполняют определённую задачу. Они помогают избежать повторений, упрощают чтение и поддержку программы.

🔧
Что такое функция?

Функция — это именованный блок кода, который можно вызывать по имени. Она может принимать входные данные (параметры) и возвращать результат.

Зачем нужны функции?

  • Избегают повторения кода (DRY — Don't Repeat Yourself).
  • Упрощают отладку и тестирование.
  • Делают код более структурированным и читаемым.
📝
Определение функции (def)

Функция определяется с помощью ключевого слова def, затем идёт имя функции, круглые скобки с параметрами и двоеточие. Тело функции пишется с отступом.

def greet():
    print("Привет, мир!")

# Вызов функции
greet()  # Выведет: Привет, мир!
📥
Параметры и аргументы

Функция может принимать параметры — переменные, которые получают значения при вызове. Эти значения называются аргументами.

def greet_user(name):
    print(f"Привет, {name}!")

greet_user("Анна")  # Привет, Анна!
📤
Возврат значения (return)

Функция может возвращать результат с помощью return. После return выполнение функции прекращается.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

Если return не указан, функция возвращает None.

🔒
Локальные переменные

Переменные, определённые внутри функции, называются локальными. Они доступны только внутри этой функции и не видны снаружи.

def my_func():
    x = 10  # локальная переменная
    print(x)

my_func()   # 10
# print(x)  # Ошибка: name 'x' is not defined
⚙️
Параметры по умолчанию

Можно задать значение параметра по умолчанию. Если при вызове аргумент не передан, используется значение по умолчанию.

def greet(name="гость"):
    print(f"Привет, {name}!")

greet("Анна")   # Привет, Анна!
greet()         # Привет, гость!
