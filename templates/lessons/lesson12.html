Строки в Python — это последовательности символов, заключённые в кавычки. Они используются для хранения текстовой информации.

📝 Создание строк

Строки можно создавать, используя одинарные или двойные кавычки:

name = 'Alice'
print(name)  # Alice
city = "New York"
print(city)  # New York
⚠️ Апострофы и кавычки внутри строк

Если строка содержит апостроф, лучше использовать двойные кавычки, или экранировать символ \':

# Используем двойные кавычки
text = "It's a beautiful day"
print(text)
# Экранирование апострофа
text = 'It\'s a beautiful day'
print(text)
➕ Сложение строк (конкатенация)

Строки можно складывать с помощью оператора +:

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)  # John Doe

Числа и строки напрямую складывать нельзя — нужно преобразовывать числа в строки функцией str():

age = 25
message = "I am " + str(age) + " years old"
print(message)  # I am 25 years old
🔢 Нумерация символов (индексы)

Каждый символ в строке имеет индекс: первый символ — индекс 0, второй — 1 и т.д.

word = "Python"
print(word[0])  # P
print(word[1])  # y
print(word[5])  # n

Можно использовать отрицательные индексы для отсчёта с конца:

print(word[-1])  # n (последний символ)
print(word[-2])  # o (предпоследний)
📏 Длина строки

Функция len() возвращает количество символов в строке:

phrase = "Hello, world!"
print(len(phrase))  # 13
