Иногда нужно прервать выполнение цикла досрочно или пропустить отдельные итерации. Для этого в Python есть операторы break, continue и специальный блок else.

⛔
Оператор break

break немедленно завершает выполнение цикла, даже если условие цикла ещё истинно или остались элементы для перебора.

for i in range(1, 10):
    if i == 5:
        break      # цикл прервётся на 5
    print(i)
# Вывод: 1 2 3 4

Часто используется для досрочного выхода при обнаружении искомого элемента.

⏭️
Оператор continue

continue пропускает оставшуюся часть текущей итерации и переходит к следующей (проверке условия для while или следующему элементу для for).

for i in range(1, 6):
    if i == 3:
        continue   # пропускаем вывод тройки
    print(i)
# Вывод: 1 2 4 5
➕
Блок else в циклах

Блок else выполняется, если цикл завершился естественным образом (без break).

numbers = [2, 4, 6, 8]
for n in numbers:
    if n % 2 != 0:
        print("Найдено нечётное число!")
        break
else:
    print("Все числа чётные")   # выполнится, так как break не сработал
🔀
Break во вложенных циклах

break прерывает только тот цикл, в котором он находится. Чтобы выйти из внешнего цикла, можно использовать флаг или исключение.

found = False
for i in range(1, 4):
    for j in range(1, 4):
        if i * j == 6:
            print(f"Найдено: {i} * {j} = 6")
            found = True
            break   # выход из внутреннего цикла
    if found:
        break       # выход из внешнего
♾️
Бесконечный цикл с break

Бесконечный цикл (while True) удобно использовать, когда условие выхода сложное или зависит от ввода пользователя.

while True:
    cmd = input("Введите команду (exit для выхода): ")
    if cmd == "exit":
        break
    print("Выполняю:", cmd)
🔎
Пример: поиск элемента в списке
students = ["Анна", "Борис", "Виктор", "Галина"]
search = "Виктор"
for student in students:
    if student == search:
        print(f"{search} найден!")
        break
else:
    print(f"{search} отсутствует в списке")

Блок else сообщает, что элемент не найден.
