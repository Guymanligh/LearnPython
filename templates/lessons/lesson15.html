Ветвление позволяет программе выбирать, какой код выполнять, в зависимости от условий. В Python для этого используются операторы if, elif и else.

🔀
Простое ветвление: if

Конструкция if проверяет условие и, если оно истинно, выполняет блок кода.

temperature = 25
if temperature > 20:
    print("На улице тепло")

Обратите внимание на двоеточие в конце строки с if и отступы внутри блока.

🔁
Альтернатива: if-else

else добавляет блок, который выполняется, если условие ложно.

age = 16
if age >= 18:
    print("Доступ разрешён")
else:
    print("Доступ запрещён")
📊
Несколько условий: if-elif-else

elif (сокращение от else-if) позволяет проверить несколько условий подряд.

score = 85
if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"

print(f"Оценка: {grade}")

Условия проверяются сверху вниз; выполнится только первый истинный блок.

📦
Вложенные условия

Один условный оператор можно поместить внутрь другого.

num = 10
if num >= 0:
    if num == 0:
        print("Число равно нулю")
    else:
        print("Число положительное")
else:
    print("Число отрицательное")
❔
Тернарный оператор (условное выражение)

Короткая запись if-else в одну строку.

age = 20
status = "взрослый" if age >= 18 else "несовершеннолетний"
print(status)  # взрослый
