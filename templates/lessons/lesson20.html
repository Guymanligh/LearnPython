Список (list) — это упорядоченная коллекция элементов, которая может содержать объекты любых типов. Списки изменяемы, их можно расширять, сокращать и модифицировать.

📋
Создание списка

Список создаётся с помощью квадратных скобок [], элементы перечисляются через запятую.

# Пустой список
empty = []

# Список чисел
numbers = [1, 2, 3, 4, 5]

# Смешанный список
mixed = [10, "hello", True, 3.14]

# Список списков (вложенный)
matrix = [[1, 2], [3, 4], [5, 6]]
🔢
Доступ по индексу

Элементы списка нумеруются с нуля. Можно использовать отрицательные индексы для доступа с конца.

fruits = ["яблоко", "банан", "апельсин"]
print(fruits[0])    # яблоко
print(fruits[2])    # апельсин
print(fruits[-1])   # апельсин (последний)
print(fruits[-2])   # банан
✏️
Изменение элементов

Списки изменяемы, поэтому можно присваивать новое значение по индексу.

colors = ["red", "green", "blue"]
colors[1] = "yellow"
print(colors)   # ["red", "yellow", "blue"]

🔪
Срезы (slicing)

Срезы позволяют получить подсписок.

nums = [0, 1, 2, 3, 4, 5]
print(nums[1:4])   # [1, 2, 3]  (от индекса 1 до 3 включительно)
print(nums[:3])    # [0, 1, 2]  (от начала до 2)
print(nums[3:])    # [3, 4, 5]  (от 3 до конца)
print(nums[::2])   # [0, 2, 4]  (каждый второй)
print(nums[::-1])  # [5, 4, 3, 2, 1, 0] (разворот)
➕
Операции со списками
# Конкатенация (+)
a = [1, 2]
b = [3, 4]
c = a + b        # [1, 2, 3, 4]

# Повторение (*)
d = [0] * 3      # [0, 0, 0]

# Проверка вхождения (in)
if 2 in a:
    print("2 есть в списке")

# Длина списка
print(len(a))    # 2
