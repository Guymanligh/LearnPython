Словари и кортежи — это важные структуры данных в Python. Словари хранят пары «ключ — значение», а кортежи — неизменяемые последовательности.

📖
Что такое словарь (dict)?

Словарь — это неупорядоченная коллекция элементов, где каждый элемент имеет ключ и значение. Ключи должны быть уникальными и неизменяемыми (числа, строки, кортежи).

# Создание словаря
person = {
    "name": "Анна",
    "age": 25,
    "city": "Москва"
}
🔑
Доступ и изменение элементов
# Доступ по ключу
print(person["name"])   # Анна
print(person.get("age")) # 25 (безопасный способ)

# Добавление новой пары
person["job"] = "программист"

# Изменение значения
person["age"] = 26

# Удаление элемента
del person["city"]
age = person.pop("age")  # удаляет и возвращает значение
🛠️
Методы словарей
МетодОписаниеПример
keys()Возвращает все ключиperson.keys()
values()Возвращает все значенияperson.values()
items()Возвращает пары (ключ, значение)person.items()
update()Обновляет словарь другим словарёмperson.update({'age': 30})
clear()Очищает словарьperson.clear()
🔄
Перебор словаря
# Перебор ключей
for key in person:
    print(key, person[key])

# Перебор значений
for value in person.values():
    print(value)

# Перебор пар ключ-значение
for key, value in person.items():
    print(f"{key}: {value}")
📦
Что такое кортеж (tuple)?

Кортеж — это упорядоченная неизменяемая последовательность элементов. Создаётся с помощью круглых скобок.

# Создание кортежа
point = (10, 20)
colors = ("red", "green", "blue")
single = (42,)  # кортеж из одного элемента (запятая обязательна)

# Без скобок (тоже кортеж)
numbers = 1, 2, 3
⚖️
Кортежи vs списки
  • Кортежи неизменяемы (нельзя добавить, удалить, изменить элемент).
  • Занимают меньше памяти и работают быстрее.
  • Могут использоваться как ключи в словарях (если содержат только неизменяемые элементы).
  • Защищают данные от случайного изменения.
🎁
Распаковка кортежей
point = (10, 20)
x, y = point   # x = 10, y = 20

# Распаковка с остатком
numbers = (1, 2, 3, 4, 5)
first, *rest = numbers  # first = 1, rest = [2,3,4,5]

# Обмен значений без временной переменной
a, b = 1, 2
a, b = b, a   # a = 2, b = 1
