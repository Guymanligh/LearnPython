В Python используются данные разных типов. Обычно Python сам понимает, какой тип использовать, но иногда нужно вручную менять один тип данных на другой.

🔢 Числа: int (целые)

Целые числа — это числа без дробной части. Они хороши для подсчёта количества. 

sheep = 1
print(sheep)  # 1
🥧 Числа: float (вещественные)

Вещественные числа записываются с десятичной точкой, например 1.5. Используются для измерений. 

sheep = 1.5
print(sheep)  # 1.5
📝 Строки (str)

Строка — это фрагмент текста, заключённый в кавычки (чаще всего одинарные). Может содержать буквы, цифры и символы. 

a = 'Coding is fun!'
print(a)  # Coding is fun!

Использование: чтобы дать переменной строковое значение, укажи текст в кавычках.

✅ Булевы значения: True
a = True
print(a)  # True
❌ Булевы значения: False
a = False
print(a)  # False

Булев тип (bool) может принимать только True или False. Эти слова пишутся с заглавной буквы без кавычек.

🔄 Преобразование типов данных

В переменных могут храниться значения любого типа. Однако не все типы данных в программе совместимы. Иногда один тип нужно преобразовать в другой, иначе возникнет ошибка. 

apple = input('Enter number of apples: ')
# Ввод: 2
print(apple + 1)  # Ошибка! TypeError
print(int(apple) + 1)  # 3
# Преобразовали строку в целое число

Смешение типов: нельзя складывать строку и число (кроме int и float между собой). Команда int() преобразует строку в целое число, float() — в вещественное, str() — в строку.
