Взаимодействие с пользователем в Python осуществляется через ввод данных с клавиатуры и вывод информации на экран.

⌨️ Ввод данных (input)

Функция input() позволяет получить данные от пользователя. Она всегда возвращает строку.

name = input("Как тебя зовут? ")
print("Привет, " + name)

Для получения числа используйте преобразование типа:

age = int(input("Сколько тебе лет? "))
print("Через год тебе будет", age + 1)
🖨️ Вывод данных (print)

Функция print() выводит данные на экран. Можно выводить несколько значений через запятую.

print("Результат:", 42)
print("Первая строка")
print("Вторая строка")

Параметры sep и end позволяют управлять разделителями и окончанием строки:

print("a", "b", "c", sep="-")  # a-b-c
print("Hello", end=" ")
print("World")  # Hello World
🎨 Форматирование строк (f-строки)

f-строки позволяют вставлять значения переменных прямо в строку.

name = "Alice"
age = 25
print(f"Меня зовут {name}, мне {age} лет.")
