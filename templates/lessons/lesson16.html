Циклы позволяют выполнять один и тот же блок кода многократно. Это основа автоматизации повторяющихся действий в программировании.

🔄
Что такое цикл?

Цикл — это конструкция, которая повторяет блок кода несколько раз. Количество повторений может зависеть от условия или от количества элементов в коллекции.

Пример из жизни: пока не закончится список покупок, мы идём по магазину и берём следующий товар.

⏳
Цикл while

Выполняет блок кода, пока условие истинно. Условие проверяется перед каждой итерацией.

count = 1
while count <= 5:
    print(f"Итерация {count}")
    count += 1

Важно не забывать изменять переменную в условии, иначе цикл станет бесконечным.

📋
Цикл for

Используется для перебора элементов последовательности (строки, списка, диапазона чисел).

# Перебор чисел от 1 до 5
for i in range(1, 6):
    print(f"Число: {i}")

# Перебор списка
fruits = ["яблоко", "банан", "апельсин"]
for fruit in fruits:
    print(fruit)
🔁🔁
Вложенные циклы

Один цикл может находиться внутри другого. Это полезно для работы с таблицами или многомерными структурами.

# Таблица умножения от 1 до 3
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} * {j} = {i*j}")
    print("---")  # разделитель
