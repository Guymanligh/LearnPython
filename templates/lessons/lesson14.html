В программировании часто нужно принимать решения в зависимости от условий. В Python для этого используются операторы if, elif, else, а также логические операции.

🤔 Условный оператор if

Оператор if выполняет блок кода, только если условие истинно.

age = 18
if age >= 18:
    print("Вы совершеннолетний")

Можно добавить else для альтернативного действия:

if age >= 18:
    print("Вход разрешён")
else:
    print("Вход запрещён")

Несколько условий проверяются с помощью elif:

score = 85
if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
else:
    grade = "C"
🧠 Логические операции

Логические операции позволяют комбинировать условия.

ОператорОписаниеПримерРезультат
andИстинно, если оба операнда истинны(5 > 3) and (2 < 4)True
orИстинно, если хотя бы один операнд истинен(5 > 3) or (2 > 4)True
notОтрицаниеnot (5 > 3)False
 
# Пример с and
age = 25
has_license = True
if age >= 18 and has_license:
    print("Можно водить машину")

# Пример с or
day = "Saturday"
if day == "Saturday" or day == "Sunday":
    print("Выходной!")
⚖️ Операции сравнения

Операторы сравнения возвращают True или False.

ОператорЗначениеПример
==равно5 == 5 → True
!=не равно5 != 3 → True
<меньше3 < 5 → True
>больше5 > 3 → True
<=меньше или равно5 <= 5 → True
>=больше или равно5 >= 3 → True
🔍 Проверка вхождения (оператор in)

Оператор in проверяет, содержится ли элемент в последовательности (строке, списке, кортеже).

# Проверка в строке
text = "Hello world"
print('H' in text)   # True
print('x' in text)   # False

# Проверка в списке
fruits = ['apple', 'banana', 'orange']
print('banana' in fruits)  # True
print('grape' in fruits)    # False

# Проверка в словаре (по ключам)
person = {'name': 'Alice', 'age': 30}
print('name' in person)   # True
print('city' in person)    # False
🧪 Примеры совпадений

Часто используется комбинация in с условными операторами:

# Проверка наличия подстроки
message = "Ваш заказ подтверждён"
if "подтверждён" in message:
    print("Заказ успешно оформлен!")

# Проверка нескольких вариантов
color = "red"
if color in ["red", "green", "blue"]:
    print("Это основной цвет")
